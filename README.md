# JAOPuTo
This package contains a number of functions to download, transform and visualize datasets from the JAO Publication Tool.
The JAO Publication Tool is used by the Core Transmission System Operators to publish, on a daily basis, a number of datasets related to day-ahead and intraday capacity calculation (and allocation).
